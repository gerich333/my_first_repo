# Подсказки по командной строке

Комана смены директории
'''sh
cd c:\folder_name
'''

Команда отображения текущей директории. MacOs, Linux
'''sh
pwd
'''

Литинг текущей директории Windows:
'''sh
dir
'''
Linux, MacOs:
'''sh
ls
'''

Удаление файла в Windows:
'''sh
del <filename>
'''
в MacOs, Linux:
'''sh
rm <filename>
'''
